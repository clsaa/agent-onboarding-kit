# Anthropic Cybersecurity Skills

- **Category:** Security skill pack (817 skills across 29 domains)
- **What it does:** A large library of cybersecurity workflows (cloud security, threat hunting, malware analysis, incident response, red teaming) as `SKILL.md`-format skills, each mapped to industry frameworks (MITRE ATT&CK, NIST CSF 2.0, ATLAS, D3FEND, NIST AI RMF).
- **Repo:** https://github.com/mukul975/Anthropic-Cybersecurity-Skills
- **Compatibility:** Claude Code ✅ | Codex ✅ | OpenCode ✅
- **Install:**
  - Claude Code: `npx skills add mukul975/Anthropic-Cybersecurity-Skills`
  - Codex / OpenCode / others: `npx skills add mukul975/Anthropic-Cybersecurity-Skills` (follows the agentskills.io open standard; also `git clone` for manual placement)
- **Verify:** Ask the agent for a threat-hunting or incident-response workflow; a framework-mapped skill (e.g. citing MITRE ATT&CK) is applied.
- **When to use / skip:** Use for security-engineering work that benefits from standardized, framework-mapped playbooks; skip if you don't do security work or want a lean install (it's a large pack).
- **Stars:** ~25.3k (as of 2026-07)
